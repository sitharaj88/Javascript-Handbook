## 🌟 1. DOM Manipulation (Virtual DOM Concepts)

Learn how to efficiently manipulate the **DOM** and understand how the **Virtual DOM** works in frameworks like React.

### Key Concepts:
- **Selecting Elements:** `getElementById`, `querySelector`, etc.
- **Modifying Content:** Changing text, HTML, and attributes.
- **Virtual DOM:** Efficient UI rendering and reconciliation processes.
- **Performance Optimization:** Minimizing reflows and repaints.

## ⚡ 2. Events (Bubbling, Custom Events)

Understand JavaScript’s event system, including how events propagate through the DOM and how to create custom events.

### Highlights:
- **Event Listeners:** `addEventListener`, removing listeners.
- **Event Propagation:** Bubbling and capturing phases.
- **Custom Events:** Creating and dispatching custom events.
- **Debounce & Throttle:** Optimizing event handling.

Master **Web Components** for building reusable UI components with encapsulated behavior.

### Covered Topics:

## 🗄️ 4. Storage APIs (IndexedDB, Cookies)

Learn how to store and manage data on the client side using browser storage APIs.

### Key Topics:
- **Cookies:** Storing small amounts of data with expiration.
- **LocalStorage & SessionStorage:** Key-value storage for persistent and session data.
